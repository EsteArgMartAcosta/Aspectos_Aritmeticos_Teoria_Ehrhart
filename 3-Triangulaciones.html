Triangulaciones

En esta sección se presentan algoritmos para subdividir poliedros en triángulos y su aplicación en conteos de Ehrhart.

Ejemplo de código

# Ejemplo simple de triangulación en Python
import matplotlib.pyplot as plt
import matplotlib.tri as mtri
import numpy as np

x = np.random.rand(30)
y = np.random.rand(30)
triang = mtri.Triangulation(x, y)

plt.triplot(triang, 'go-')
plt.show()
