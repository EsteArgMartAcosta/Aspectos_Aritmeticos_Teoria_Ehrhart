Construcción de Gráficos

En esta sección describimos los métodos y teorías para construir gráficos que representan estructuras poligonales y funciones discretas.

Ejemplo de código

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.plot(x, y)
plt.show()
