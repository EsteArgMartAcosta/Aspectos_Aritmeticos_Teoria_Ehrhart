Envolvente Convexa

La envolvente convexa de un conjunto de puntos en el plano es la frontera poligonal m√°s peque√±a que encierra todos los v√©rtices, definiendo as√≠ el contorno del conjunto. En esta secci√≥n se exploran dos estrategias para construir dicho contorno de manera din√°mica, ilustrando cada paso mediante animaciones que muestran c√≥mo se incorporan o descartan puntos hasta completar la frontera.

  • Graham Scan: Se parte de un pivote (el punto de coordenada m√≠nima en y) y se ordena el resto seg√∫n su √°ngulo polar respecto a √©l. A medida que se recorre esta lista, se mantiene una pila de v√©rtices que definen el contorno parcial. Cada nuevo punto es comparado con los dos √∫ltimos de la pila mediante un test de orientaci√≥n; si forma un giro hacia dentro, el v√©rtice intermedio es desapilado, garantizando que la frontera permanezca convexa.
  • Quickhull: Se identifican dos puntos extremos (el de m√≠nima x y el de m√°xima x) y se traza el segmento que los une, dividiendo el conjunto en los situados por encima y por debajo de esa l√≠nea. Para cada subconjunto, se selecciona el punto m√°s alejado de la base como nuevo v√©rtice de la envolvente y se generan dos subproblemas recursivos sobre los tri√°ngulos formados, hasta cerrar el pol√≠gono.

import numpy as np
import plotly.graph_objects as go
from IPython.display import HTML

HTML('''
 Envolvente Convexa Din√°mica Avanzada
''')

np.random.seed(42)
pts = np.random.randint(0, 200, size=(200, 2))
pts_sorted = pts[np.lexsort((pts[:,1], pts[:,0]))]

def incremental_hull(points):
    def cross(o, a, b):
        return (a[0]-o[0])*(b[1]-o[1]) - (a[1]-o[1])*(b[0]-o[0])
    lower, upper, steps = [], [], []
    for p in points:
        lower.append(p.tolist())
        while len(lower)>=3 and cross(lower[-3], lower[-2], lower[-1])<=0:
            lower.pop(-2)
        steps.append(lower.copy())
    for p in points[::-1]:
        upper.append(p.tolist())
        while len(upper)>=3 and cross(upper[-3], upper[-2], upper[-1])<=0:
            upper.pop(-2)
        full = lower + upper[1:-1]
        steps.append(full.copy())
    return steps

hull_steps = incremental_hull(pts_sorted)

frames = []
for hull in hull_steps:
    hx = [pt[0] for pt in hull] + ([hull[0][0]] if hull else [])
    hy = [pt[1] for pt in hull] + ([hull[0][1]] if hull else [])
    frames.append(go.Frame(
        data=[go.Scatter(x=pts[:,0], y=pts[:,1], mode='markers',
              marker=dict(size=6, color='#4ea8ff'), name='Puntos'),
            go.Scatter(x=hx, y=hy, mode='lines+markers',
            line=dict(width=2, color='#fde725'),
            marker=dict(size=5, color='#fde725'), name='Hull')]))

fig = go.Figure(data=frames[0].data,frames=frames)
fig.update_layout(updatemenus=[dict(type='buttons',showactive=False,
        x=1.15, y=1,
        xanchor='right', yanchor='top',
        buttons=[dict(label='‚ñ∂ Play',
        method='animate',
        args=[None, {"frame": {"duration": 100, "redraw": True},
        "fromcurrent": True, "mode": "immediate"}])])],
    margin=dict(l=0, r=0, b=0, t=0),
    paper_bgcolor='black',
    plot_bgcolor='black',
    xaxis=dict(visible=False),
    yaxis=dict(visible=False),
    template='plotly_dark',
    height=600)
    fig.show()
